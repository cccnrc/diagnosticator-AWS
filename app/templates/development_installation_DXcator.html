First thing first, for the development you can choose two different ways to work on Diagnosticator:
1. through Docker containers
2. directly through Flask

The difference is just the time it takes for your changes to take place on the application and portability, the Flask-way it is straightforward and you will see changes effect immediately, through Docker you need to re-build the image and run it. On the other hand, the Flask-way requires more set-up work.

As all techies we assume you work on Linux
