Docker Installation

x
In order to install Diagnosticator locally on your machine you need to have Docker and Docker-Compose installed and running.

Once you have them running, you can simply download the latest docker-compose.yml file from our repository here

Save it on a dedicated folder in your local machine (Desktop/diagnosticator in the current example) and go to that folder from your terminal: 
        

cd ~/Desktop/diagnosticator
Then simply pull the containers and run Diagnosticator with: 
        

docker-compose up
The first time you run it, the pulling of all necessary images and building of the local application may take several minutes. Once completed, the terminal output should look something like this:

Then you can simply click here.
Otherwise go to your browser (Firefox or Chrome recommended) and copy-paste in the URL bar: http://localhost:9000

That's all! You are now ready to navigate Diagnosticator! :)
